Workout Tracker — CRUD Application (Node.js + Express + MongoDB + EJS)

This is a full CRUD web application built using Node.js, Express.js, MongoDB/Mongoose, and EJS templating.
It allows users to create, view, update, and delete workouts, with full routing, controllers, models, and views.

This project was completed for INFR3120 – Assignment 3.

Project Description

The Workout Tracker is designed to help users log workouts with the following details:

Workout Name

Duration (minutes)

Difficulty (Easy / Medium / Hard)

Date Created

The application demonstrates secure CRUD functionality, MVC structure, deployment, and styling with Bootstrap + custom CSS.

Features (CRUD)
Feature	Description
Create	Add a new workout through a form
Read	View all workouts in a table
Update	Edit an existing workout
Delete	Delete with confirmation page
Database	MongoDB Atlas using Mongoose
Templating	EJS templates with shared layout
